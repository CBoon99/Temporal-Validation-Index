TVI API Documentation

The TVI API allows you to programmatically calculate Temporal Validation Index scores for any domain.

Authentication

All API requests require an API key. Generate your key from the dashboard.

Include your API key in the X-API-Key header:

X-API-Key: tvi_your_api_key_here

Rate Limits

  • Free Plan: 100 requests per day
  • Pro Plan: 1,000 requests per day
  • Enterprise: Custom limits (contact us)

Endpoints

POST /api/tvi

Calculate TVI Score

Compute TVI for any domain with custom inputs.

Request Body

{
  "domain": "viral",
  "inputs": {
    "year": 2007,
    "views": 880000000,
    "users": 200000000,
    "c": 3.0,
    "p": 180,
    "r": 0.5,
    "legacy": 2.5
  }
}

Response

{
  "score": 63.4,
  "type": "TVI",
  "classification": "Historical Phenomenon",
  "components": [
    { "label": "CSI", "value": "10.56" },
    { "label": "TVS", "value": "252.0" },
    { "label": "SRC", "value": 2.5 }
  ],
  "timestamp": "2026-01-15T12:00:00.000Z"
}

Domains

  • viral - Cultural media (requires: year, views, users, c, p, r, legacy)
  • datasets - AI datasets (requires: year, usage, researchers, c, p, r, legacy)
  • business - Business methods (requires: year, adoption, market, c, p, r, legacy)
  • invest - Investments (requires: year, brand, position, era, sus, crisis, leadership)

Example: cURL

curl -X POST https://tvi.boonmind.com/api/tvi \\
  -H "Content-Type: application/json" \\
  -H "X-API-Key: tvi_your_api_key" \\
  -d '{
    "domain": "datasets",
    "inputs": {
      "year": 1998,
      "usage": 50000000,
      "researchers": 10000,
      "c": 3.5,
      "p": 180,
      "r": 0.9,
      "legacy": 3
    }
  }'
